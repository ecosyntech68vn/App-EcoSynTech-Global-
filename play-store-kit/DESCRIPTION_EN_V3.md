## Full description

🌱 **EcoSynTech Farm OS V3.0 — Comprehensive farm operating system**

V3.0 leaps from "monitoring app" to a real OS for small-to-mid farms. **18 integrated features**, offline-first, frugal, human-centric.

**18 features:**

📊 **Core (V1.0)** — PIN/Biometric login · Real-time Sensor Dashboard · Alert Center · Operation Log · Task List · Flexible LAN/Cloud Settings

⚡ **Smart Operations (V1.1)** — 24/7 Push notifications · 2-step device control (pump/light/fan) · Camera + GPS metadata · Auto irrigation/fertilizer schedules · Sensor history charts 24h/7d/30d

🧠 **Smart Tools (V1.2)** — Biometric/Fingerprint login · Device QR scan · Weather forecast (OpenWeather) · In-app OTA from GitHub Releases · 15-min background sync

🏆 **Pro Features (V1.3)** — Multi-role (Farmer/Manager/Tech) · Multi-farm switcher · Weekly/Monthly PDF reports · Activity photo timeline · Inline SOP viewer · Pest log with severity 1-5 · Zalo share · Geofence auto check-in

**Philosophy:** Safety → Reliability → Recovery → Simplicity → Cost. Strict offline-first. Frugal bundle <800KB. Human-centric UI with one-hand operation.

**No ads. No trackers. No data selling.** App communicates only with your server (WLC LAN + your GAS Cloud). EcoSynTech Global receives no data.

**Requirements:** Android 8.0+ (API 26), WLC v6.0 or GAS V10.3 backend.
