## Mô tả đầy đủ

🌱 **EcoSynTech Farm OS V3.0 — Hệ điều hành nông nghiệp toàn diện**

Phiên bản 3.0 là bước nhảy lớn từ "app theo dõi" sang "hệ điều hành thực sự" cho nông trại quy mô vừa và nhỏ. **18 tính năng tích hợp**, vận hành offline-first, frugal, human-centric.

**18 tính năng:**

📊 **Cốt lõi (V1.0)**
- Đăng nhập PIN/Biometric · Sensor Dashboard real-time · Alert Center · Nhật ký vận hành · Task List · Cài đặt linh hoạt LAN/Cloud

⚡ **Smart Operations (V1.1)**
- **Push notification 24/7** — nhận cảnh báo ngay cả khi app đóng
- **Điều khiển thiết bị 2-step** — bật/tắt bơm, đèn, quạt với xác nhận PIN
- **Camera + GPS metadata** — ảnh có toạ độ truy xuất nguồn gốc
- **Lịch tưới/bón tự động** — schedule cron-like
- **Biểu đồ lịch sử 24h/7d/30d** — sensor trend

🧠 **Smart Tools (V1.2)**
- **Đăng nhập vân tay/Face ID** — nhanh, an toàn
- **QR scan thiết bị** — link nhanh thiết bị → lệnh
- **Thời tiết** — forecast 24h từ OpenWeather
- **Cập nhật in-app** — OTA từ GitHub Releases
- **Background sync 15 phút** — không lo mất data offline

🏆 **Pro Features (V1.3)**
- **Đa vai trò** — Farmer / Manager / Tech với quyền riêng
- **Đa nông trại** — switcher 1 chạm
- **Báo cáo PDF tuần/tháng** — generate offline, share
- **Album hoạt động** — photo timeline theo ngày + zone
- **SOP inline** — quy trình chuẩn xem ngay trong app
- **Báo sâu bệnh** — form severity 1-5 với ảnh
- **Chia sẻ Zalo** — alert deeplink
- **Geofence auto check-in** — tự ghi nhận khi đi vào zone

**Triết lý**
- An toàn → Tin cậy → Phục hồi → Đơn giản → Chi phí
- Offline-first nghiêm ngặt — mọi feature work khi mất sóng
- Frugal — bundle <800KB, không bloat
- Human-centric — UI 1 tay, font lớn, đa số thao tác ≤4 tap

**Không quảng cáo. Không tracker. Không bán dữ liệu.**

App chỉ giao tiếp với server riêng (WLC LAN + GAS Cloud do bạn deploy). EcoSynTech Global không nhận data của bạn.

**Yêu cầu:**
- Android 8.0+ (API 26)
- WLC v6.0 hoặc GAS V10.3 trên backend nông trại
